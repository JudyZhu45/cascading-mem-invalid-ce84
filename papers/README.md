3. **[MemGPT: Towards LLMs as Operating Systems](2310.08560_memgpt_llm_os.pdf)**
   - Authors: Packer, Wooders, Lin, Fang, Patil, Stoica, Gonzalez (UC Berkeley)
   - Year: 2024
   - arXiv: 2310.08560
   - Why relevant: Hierarchical memory architecture (working context + recall storage + archival storage) via OS-inspired virtual context management. Foundational for memory system design.

4. **[PersonaMem-v2: Implicit User Personas and Agentic Memory](2512.06688_personamem_v2.pdf)**
   - Authors: Jiang, Yuan, Shen et al. (UPenn, MIT, UW)
   - Year: 2025
   - arXiv: 2512.06688
   - Why relevant: State-of-the-art personalization dataset with 20K+ preferences and agentic memory framework. Models dynamic preference evolution.

5. **[HorizonBench: Long-Horizon Personalization with Evolving Preferences](2604.17283_horizonbench_evolving_prefs.pdf)**
   - Authors: Li, Paranjape, Oktar et al. (UW, Meta, OpenAI, Stanford)
   - Year: 2026
   - arXiv: 2604.17283
   - Why relevant: **Most directly relevant to research hypothesis.** Implements typed dependency edges for preference evolution, diagnoses belief-update failure as distinct from retrieval failure. Pre-evolution hard-negative distractors test cascading preference invalidation.
